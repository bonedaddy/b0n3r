# b0n3r

Erecting I2P tunnels for anonymous microservice development, with fully decentralized ddos mitigation using VDF captchas.

# features

* Complete control over I2CP tunnel settings
* VDF based ddos/flood control
* Multi Protocol:
  * TCP - done
  * UDP - todo
